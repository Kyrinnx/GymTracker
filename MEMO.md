# GymTracker — Mémo

## Mettre à jour l'app

1. Terminal : `cd ~/Desktop/ANTIGRAVITY/GymTracker && ./update.sh`
2. iPhone Safari : tape l'URL affichée → Ouvrir avec AltStore
3. Terminal : Ctrl+C

Pas besoin de câble USB — tout passe en WiFi.

## Si ça bloque à l'installation

Si AltStore dit « serveur introuvable » :
1. Sur ton Mac : quitte AltServer (losange ⬩ → Quit) puis relance AltServer + ouvre Mail.app
2. Sur ton iPhone : ouvre AltStore → My Apps → Refresh All
3. Retente l'installation via Safari

## Mes données

Tes sauvegardes sont automatiques et vont sur **iCloud Drive** chaque jour quand tu ouvres l'app. Visible depuis ton Mac dans Finder → iCloud Drive → Sauvegardes GymTracker. Tu n'as rien à faire.

Pour restaurer : dans l'app → Réglages → Données → Importer un fichier

## L'app expire ?

AltStore renouvelle tout seul si AltServer tourne sur ton Mac. Si l'app ne s'ouvre plus : ouvre AltStore → Refresh All.

## Modifier l'app

Ouvre Claude Code et dis « On reprend GymTracker » puis `./update.sh` quand c'est fini.
